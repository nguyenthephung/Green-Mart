# 🧑‍💻 Dự án Web Fullstack

Dự án gồm 2 phần: **frontend** (Vite + React + TypeScript) và **backend** (Node.js + Express).

## 📂 Cấu trúc thư mục

```bash
.
├── frontend/       # Giao diện người dùng
├── backend/        # API và xử lý server

Chạy riêng Frontend
cd frontend
npm install
npm run dev

Chạy riêng Backend
cd backend
npm install
npm run dev

Chạy cả 2
npm install
npm run dev
