### 🔧 **API Environment Variables Setup**

The application requires several API keys for full functionality:

| Service | Required Variables | Purpose |
|---------|-------------------|---------|
| **Cloudinary** | `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, `CLOUDINARY_API_SECRET` | Image upload and management |
| **MoMo Payment** | `MOMO_ACCESS_KEY`, `MOMO_SECRET_KEY`, `MOMO_PARTNER_CODE` | Payment processing |
| **Email Service** | `SMTP_HOST`, `SMTP_USER`, `SMTP_PASS` | Order confirmations and notifications |
| **Database** | `MONGODB_URI` | Data storage |
| **Authentication** | `JWT_SECRET` | Secure user sessions |

> 💡 **Tip:** For development, you can use MongoDB Atlas (free tier) for database and Cloudinary (free tier) for image storage.
